# Resume Repository

This repository contains the code for displaying a resume in HTML format.

## Usage

To view the resume, open the `index.html` file in any web browser. The resume will be displayed within an iframe.

## File Structure

- `index.html`: HTML file containing the code for displaying the resume.
- `resume.pdf`: PDF version of the resume.

## Preview

The resume is embedded within an iframe and can be viewed directly in the browser.
